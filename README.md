## Aide-mémoire
### Règles d'Or pour l'Equipe
- Toujours pull avant de push
- Messages de commit clairs : "feat: ajout fonction afficher_menu"
- Branches descriptives : feature-menu, fix-bug-affichage
- Communication : Parler avant de merger
- Code reviews : Toujours 1 autre membre approuve
